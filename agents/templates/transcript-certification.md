# Transcript certification

Use at end of F3+ deliberation transcripts. See **`checklists/courtroom-scribe.md`** for full Scribe checklist.

## Certification line

Append to transcript:

```markdown
---

> *Transcript certified by MORNINGSTAR::SCRIBE*
```

## Optional closing block

If the transcript does not already end with the certification line, ensure the closing is:

```markdown
---

> *Transcript certified by MORNINGSTAR::SCRIBE*
```

No other certification format. Single canonical line.
