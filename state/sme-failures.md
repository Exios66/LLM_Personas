# SME Failure Log

> *When expert input leads to poor outcomes, the court records it here for institutional learning.*

This file is referenced by [`core/sme-framework.md`](../core/sme-framework.md). Add an entry whenever SME (Witness or Specialist) testimony or participation contributed to a suboptimal ruling, implementation, or outcome.

---

## How to Add an Entry

Use the following structure. Copy the template below, fill it in, and append to the table.

| Field | Description |
|-------|-------------|
| **Date** | YYYY-MM-DD |
| **Domain** | e.g. security, database, compliance |
| **SME Type** | Witness \| Specialist |
| **Matter** | Brief description of the deliberation or task |
| **What went wrong** | How the SME input led to a poor outcome |
| **Lesson** | What to do differently next time |
