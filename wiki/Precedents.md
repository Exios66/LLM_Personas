# Precedents

The precedent database is the authoritative index of precedent-setting decisions. Consult before deliberating on familiar matters.

**Location in repo:** `courtroom/precedents.md`

## Contents

- **Master index** — Case ID, date, matter, ruling, vote, implications, status.
- **Index by category** — e.g. Infrastructure (INFRA), Architecture (ARCH).
- **Citation format** — How to cite in deliberation, transcripts, and documentation.
- **Shorthand** — e.g. `Per 2026-INFRA-001-001`, `See Case 2026-INFRA-001-001`.
- **Full entries** — For significant rulings: Matter, Ruling, Rationale, Risk, Implications, Key quotes, Related precedent.

## Precedent Weight

| Status | Meaning |
|--------|---------|
| **BINDING** | Must be followed unless overruled |
| **PERSUASIVE** | Should be considered |
| **DISTINGUISHED** | Applied to different facts |
| **OVERRULED** | Explicitly reversed |
| **RECONSTRUCTED** | Transcript recovered, not original |
